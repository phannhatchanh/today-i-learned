# TypeScript - Utilities

## Record

```ts
type RegisterFormInput = {
  name: string
  email: string
  password: string
  address: string
  phoneNumber: string
  username: string
}
```

can be written like this:

```ts highlight="10"
enum RegisterFormField {
  name,
  email,
  password,
  address,
  phoneNumber,
  username,
}

type RegisterFormInput = Record<keyof typeof RegisterFormField, string>
```

or :

```ts highlight="12,13,14,16"
enum RegisterFormField {
  name,
  email,
  password,
  address,
  phoneNumber,
  username,
}

type RegisterFormFieldType = keyof typeof RegisterFormField

type RegisterFormInput = {
  [key in RegisterFormFieldType]: string
}
/** or */
type RegisterFormInput = Record<RegisterFormFieldType, string>
```

other examples:

```ts highlight="1,7,20"
type CardSpecs = {
  name: string
  ID: number
  isPassive: boolean
}

type CardRarity = 'common' | 'rare' | 'epic'
/**
can be written like this:

enum CardRarityName {
  common,
  rare,
  epic
}

type CardRarity = keyof typeof CardRarityName;
*/

type CardCollection = Record<CardRarity, CardSpecs>
/**
result:

type CardCollection = {
  common: {
    name: string;
    ID: number;
    isPassive: boolean;
  };
  rare: {
    name: string;
    ID: number;
    isPassive: boolean;
  };
  epic: {
    name: string;
    ID: number;
    isPassive: boolean
  }
}
*/
```

## Partial

```ts
type RegisterFormInput = {
  name: string
  email: string
  password: string
  address: string
  phoneNumber: string
  username: string
}

type OptionalRegisterFormInput = {
  name?: string
  email?: string
  password?: string
  address?: string
  phoneNumber?: string
  username?: string
}
```

can be written like this:

```ts highlight="10"
type RegisterFormInput = {
  name: string
  email: string
  password: string
  address: string
  phoneNumber: string
  username: string
}

type OptionalRegisterFormInput = Partial<RegisterFormInput>
```

reference: [TypeScript Docs](https://www.typescriptlang.org/docs/handbook/utility-types.html#partialtype)

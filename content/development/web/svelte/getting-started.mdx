# Svelte - Getting Started

```bash
npx degit sveltejs/template <project_name>
cd <project_name>

npm i
npm run dev
```

reference: [svelte.dev](https://svelte.dev/blog/the-easiest-way-to-get-started#2_Use_degit)

### sveltekit (beta)

```bash
mkdir <app_name>
cd <app_name>
npm init svelte@next

# install dependencies
npm install

# start dev server and open a browser tab
npm run dev -- --open
```

reference: [https://kit.svelte.dev/](https://kit.svelte.dev/)
